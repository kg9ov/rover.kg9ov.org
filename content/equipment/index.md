The rover is all-mode capable on all amateur bands from HF thru 1296MHz with an output of at least 100w on all bands.  The main operating position is fully computer/software controlled and remote capable.  

#### Radios
- FlexRadio FLEX-6600
- Q5 Signal 5BVUX 5-Band VHF/UHF Transverter
- Q5 Signal REF10-4 10MHz Oscillator and Signal Divider

#### Amplifiers
- HF/50 - Elecraft KPA500 / KAT500 
- 144 - TE Systems 1452G 
- 222 - TE Systems 2252G 
- 432 - TE Systems 4452G 
- 902 - Converted Motorola cellular amp
- 1296 - SSB Electronic TLA1275-100MC

#### Power
- Homebrew 12v 628Ah LiFePO4 battery pack
  - 8x EVE MB31 LiFePO4 cells
  - JKBMS JK-B2A8S20P
  - Sun Fun Kits cell heating pads
  - Homebrew ESP based monitor and heater controller
- Victron BMV-712 Battery Monitor
- 2x Victron Blue Smart IP22 30A chargers
- 2x Victron Orion XS 12/12-50A DC-DC in Power Supply mode
- Victron Phoenix Inverter Smart 12/2000
- Champion 2500w Inverter Generator

#### Compute & Network
- Intel NUC11PAHi5 / 32GB RAM / 500GB NVMe SSD
- 2x ASUS PA278CV 27" Monitors
- Raspberry Pi 3 Model B (NTP / GPS / APRS / NodeRed)
- GlobalSat MR-350P-S4 GPS
- Raspberry Pi 4 Model B (Station / Rotor control)
- ASUS VT168HR 15" Touch Monitor
- MikroTik hAP ax2 WiFi AP / Router
- Netgear GS108 Unmanaged Switch
- Starlink Mini in Veritas Vans mount
